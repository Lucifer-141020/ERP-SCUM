## 二、本次执行内容

### B-01：后端新增公开字段白名单

**文件**：`backend/server.js`
**位置**：第 198-227 行

新增 `PUBLIC_CONFIG_KEYS` 白名单（9 个字段）：
```javascript
const PUBLIC_CONFIG_KEYS = [
  'heroImages', 'serverInfo', 'playItems', 'updates',
  'siteSections', 'homeStats', 'homeFeatures',
  'serverRules', 'buildingTemplates'
];
```

`GET /api/site-config` 改为通过 `buildPublicConfig()` 只返回白名单字段。

### B-02：管理接口加鉴权

**文件**：`backend/server.js`
**位置**：第 229-236 行

新增 `GET /api/admin/site-config`，需 `Bearer` token，未登录返回 401。返回完整配置。

### B-03：前端敏感数据延迟加载

**文件**：`erp14-server-showcase.html`

| 新增函数 | 作用 | 调用时机 |
|----------|------|----------|
| `applyPublicBackendConfig(config)` | 只合并 9 个公开字段 | 未登录访客 |
| `applyFullBackendConfig(config)` | 合并全部字段（含敏感） | 管理员登录后 |
| `loadPublicBackendConfig()` | 调 `/api/site-config` 拉公开数据 | 页面初始化 |
| `loadFullBackendConfig()` | 调 `/api/admin/site-config` 拉完整数据 | 登录成功后 / 已有 token 时 |

**初始化逻辑改动**（第 5824-5829 行）：
```javascript
loadPublicBackendConfig();        // 所有访客都调
if (adminToken) {                 // 若已登录则额外拉完整数据
  loadFullBackendConfig();
}
```

**登录成功逻辑改动**（第 4686 行）：登录成功后调 `loadFullBackendConfig()` 拉取完整数据。
