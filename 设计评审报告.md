## 五、改动注意事项

| 事项 | 说明 |
|------|------|
| ⚠️ 不要改 `发布区/` | 里面的归档副本是历史快照，改了不生效 |
| ⚠️ CSS 选择器重复是有意的 | 主题覆盖/暗色/移动端的覆盖层，删除前必须确认 |
| ⚠️ `backend/data/` 不提交 Git | 运营数据只在服务器本地保存 |
| ⚠️ `backend/uploads/` 不提交 Git | 上传图片只在服务器本地保存 |
| ⚠️ 真实密码不写进代码 | 用 `ADMIN_PASSWORD` 环境变量覆盖默认密码 `admin123456` |
| ✅ 数据改动走后台 | 运营内容（文字/图片）通过后台管理面板修改，自动写入 JSON |
| ✅ 代码改动走源文件 | 样式/结构/逻辑改动直接编辑 `erp14-server-showcase.html` |
